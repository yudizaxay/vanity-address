# vanity-address (npm)

CLI wrapper that installs the native **vanity-address** binary for your platform.

```bash
npx vanity-address
npm install -g vanity-address
vanity-address --chain sol --suffix axay
```

This package does **not** compile Rust. Binaries come from [GitHub Releases](https://github.com/yudizaxay/vanity-address/releases) via platform packages (`vanity-address-darwin-arm64`, etc.).

Full docs: https://github.com/yudizaxay/vanity-address#readme
